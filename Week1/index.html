Project Name: WorldSurf

WorldSurf.com will be a cross-platform web 2.0 site where users can easily find content pertaining to different surfing locations around the world. Detailed information of each destination will be available. Each destination will include “members only” content, providing WorldSurf.com members with bookmark capability, web cam access to the destination, as well as user-supplied “tips & tricks”. WorldSurf's web application will incorporate a sign-up section that will allow users to create a free account. With an account, members will have access to the “members only” content for each destination, as well as access to “My World”, the WorldSurf.com content management systems. Here users will be able to add and edit information and photos pertaining to a destination, as well as create new surf destinations of their own (which will be moderated). WorldSurf.com will be the only site of its kind with content created for surfers, by surfers.
